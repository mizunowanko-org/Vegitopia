# 『ベジ・ウォーズ：大地の防衛線』仕様書 (Specification)

## 1. プロジェクト概要
- **ジャンル**: 2D グリッドベース リアルタイム戦略ゲーム (RTS)
- **コンセプト**: 戦うお野菜。プレイヤーは農地を広げ、自律的な野菜ユニットを育成して害虫と戦う。
- **視点**: トップダウン

## 2. 地形 (Terrain)

### 2.1 地形種別
- **川 (River)**: 進入不可・建築不可。
- **原野 (Wilderness)**: 初期地面。
- **石灰岩 (Limestone)**: 進入不可。障害物。

### 2.2 建築可能範囲
- 川のタイルから距離5マス以内の「原野」のみ「畑」を建築可能。

## 3. コアメカニクス

### 3.1 建築 (Building)
- **プロセス**: 建築可能範囲に地点を指定。建築者が対象地点に滞在して作業を継続することで進捗（Progress）が上昇。建築者が不在の間、進捗は停止する。進捗が最大に達することで「畑」が成立。
- **修理**: 建築者がダメージを受けた「畑」に滞在することで、耐久度が回復。

### 3.2 農耕 (Farming)
- **プロセス**: 「種」を畑に蒔く。農耕者が対象の畑に滞在している間のみ、成長タイマーが進行。農耕者が不在の間、成長は停止する。一定時間の経過後、野菜ユニットが生成される。

### 3.3 戦闘 (Combat)
- **自律AI**: 全ユニットは自律的に行動する。
- **敵の行動**: 「畑」を優先的な攻撃対象とし、経路上の野菜ユニットに対しても攻撃を行う。

## 4. ユニット定義

### 4.1 野菜（味方）
| 役割 | 野菜 | 特性 |
| :--- | :--- | :--- |
| **農耕者** | **ダイコン** | 畑に滞在し、野菜の成長を進行させる。 |
| **建築者** | **ジャガイモ** | 指定地点に滞在し、建築および修理を進行させる。 |
| **戦闘者** | **トウガラシ** | 敵ユニットを優先的に追跡し、攻撃を行う。 |

### 4.2 害虫（敵）
| 名称 | 特徴 |
| :--- | :--- |
| **アブラムシ** | 低速で移動し、隣接する「畑」または野菜ユニットにダメージを与える。 |
